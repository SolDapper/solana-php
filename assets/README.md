# Assets

Brand assets for Solana PHP.

## Files

| File | Size | Format | Use |
|------|------|--------|-----|
| `logo.jpg` | 832×832 | JPEG | Master source file. Use this if you need the original, or as a README hero image. |
| `logo-512.png` | 512×512 | PNG | GitHub social preview (GitHub requires a minimum of 640×640, so this may need upscaling for that specific use case). |
| `logo-256.png` | 256×256 | PNG | General web use. |
| `logo-128.png` | 128×128 | PNG | App icon, large favicon. |
| `logo-64.png` | 64×64 | PNG | Small favicon, list thumbnails. |

## Usage

Reference in the project README:

```markdown
<p align="center">
  <img src="assets/logo-256.png" alt="Solana PHP" width="180" />
</p>
```

Reference from HTML:

```html
<link rel="icon" type="image/png" sizes="64x64" href="/assets/logo-64.png">
<link rel="icon" type="image/png" sizes="128x128" href="/assets/logo-128.png">
```

## Provenance

The logo depicts a dark photorealistic elephant rendered in the Solana brand palette (purple to teal) against a subtle circuit-pattern background. The image was generated via AI image tooling by the project maintainer for use as the project's brand asset.

The elephant motif is a nod to PHP's longstanding mascot tradition (the ElePHPant). The color palette references Solana's brand guidelines without copying any trademarked Solana mark or logo.
